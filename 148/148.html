Kontynuacja projektowania strony z kursu. Jest kilka rzeczy, które już zapomniałem:

  • ustawianie relatywnej wielkości czcionki
  • ustawienie space-evenly w nagłówku strony
  • przestrzeń między block
  • wypełnienie obszaru przez zdjęcie
    • .two-columns-layout img { justify-self: end; } .two-columns-layout img:first-child { justify-self: start; }

Mniej więcej już kumam jak działa ten CSS, teraz trzeba praktykować w projektach, by nie wpaść w tutorial hell.
